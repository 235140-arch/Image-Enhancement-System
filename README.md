Digital Image Processing (DIP) Enhancement System

An integrated Python-based system for digital image analysis, transformation, and automated enhancement.



Project Structure

The repository is organized as follows:



code/: contains the modular lab functions.



&#x09;lab\_functions.py: Individual implementations for Labs 1–6.



&#x09;main.py: The primary execution script that runs the entire pipeline.



images/:



&#x09;input/: Original raw images (image1.jpg to image6.jpg).



&#x09;output/: Processed results from each lab stage.



DIP\_System\_Notebook.ipynb: A Jupyter Notebook containing all code, visualizations, and step-by-step execution.



Report.pdf: Comprehensive technical documentation including methodology and analysis.



What's in the Report?

The included Report.pdf provides a deep dive into the engineering behind the code:

1. Objective: Goals of the enhancement system.
2. Methodology: Phase-wise explanation of the DIP pipeline.
3. Code Snippets: Highlights of the core mathematical logic (Gamma, Log, Quantization).
4. I/O Gallery: Before-and-after visual comparisons for all 6 labs.
5. Observations \& Analysis: Technical breakdown of how bit-depth, sampling, and histograms affect image quality.
6. Conclusion: Summary of the "Best-Fit" enhancement strategy.
